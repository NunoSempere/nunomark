# Marknuno

A simple markdown to html compiler.

Architecture: a series of passes, character by character.

## Passes

- [x] Headers
- [~] Paragraphs
- [ ] Links
- [ ] Images
- [ ] Footnotes
- [ ] Bold 
- [ ] Underline
- [ ] ---
- [ ] etc.
- [ ] Checklists
